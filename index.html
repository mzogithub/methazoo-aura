methazoo-oura

personal project

a personal integration that syncs oura ring biometric data to a local smart home assistant for private use.

privacy policy

last updated: march 4, 2026

methazoo-oura is a personal-use application. it accesses oura ring data solely for the account owner's private use.

  • data collected: sleep, activity, readiness, heart rate, stress, spo2, workout, and other biometric data available through the oura api v2.
  • storage: all data is stored locally on the user's own device in an encrypted sqlite database. no data is transmitted to third-party servers.
  • sharing: no oura data is shared with, sold to, or disclosed to any third party, advertiser, or data broker.
  • retention: data is retained locally until the user deletes it.
  • deletion: users can delete all stored data at any time by removing the local database file.
  • security: oauth tokens are stored with restricted file permissions. all api communication uses https.

this application does not collect data from any user other than the account owner. for questions, contact beyondmzo@gmail.com.
